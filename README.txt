# Customer Churn Prediction

## About
Predicted customer churn using Machine Learning with 85.58% accuracy.

## What I Did
- Loaded and explored 2,666 customer records
- Built prediction model using LogisticRegression
- Created 3 customer segments using KMeans
- Achieved 85.58% accuracy

## Tools Used
- Python (Pandas, Scikit-learn)
- Machine Learning (LogisticRegression, KMeans)
- Data Visualization (Matplotlib)

## Results
- Model Accuracy: 85.58%
- Churn Rate: 26.2%
- 3 Customer Segments Identified

## Files
- `churn_analysis.ipynb` - Complete code & analysis
- `churn-bigml-80.csv` - Dataset (2,666 customers)
